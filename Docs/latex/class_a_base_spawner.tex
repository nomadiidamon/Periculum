\doxysubsection{Detailed Description}
\doxylink{class_a_base_spawner}{ABase\+Spawner} is an actor that can spawn other actors based on different spawn modes and location modes. 

It supports spawning actors in a sphere, box, capsule, or random mode, and can be customized for specific spawn behavior.
