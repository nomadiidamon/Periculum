\doxysubsection{Detailed Description}
A UObject controller that manages all gameplay message subscriptions and broadcasts for a given context object. 

Owns all listener handles and cleans them up automatically when Stop\+Listening\+To\+All is called. Intended to be owned by a \doxylink{class_u_messaging_component}{UMessaging\+Component} or a UUser\+Widget — the owner provides world context and GC rooting.
